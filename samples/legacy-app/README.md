# Módulo `:app` — referência Android (não KMP)

Esta aplicação é um **cliente Android clássico** (Activities, View system, parte em Jetpack Compose, código Java e Kotlin) usada para **demos, benchmarks e regressões** da biblioteca no ecossistema Android tradicional.

Não é um módulo **Kotlin Multiplatform** nem a amostra **Compose Multiplatform** oficial do repositório.

- Para uma stack **só CMP / KMP**, usa **`:composeApp`** (código partilhado) + **`:androidApp`** (APK de entrada).
- Coordenadas e API KMP: raiz do repositório e [MIGRATION_KMP.md](../../MIGRATION_KMP.md).
